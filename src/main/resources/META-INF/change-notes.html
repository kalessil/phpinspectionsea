Bug-fix and new features release:

- Static lambdas binding: new patterns
- Suspicious assignments: new patterns

- Callable parameter usage violates definition: resolved a false-positive (multiple cases)
- Cryptographically secure randomness: resolved a false-positive (openssl_random_pseudo_bytes signature change since 7.4)
- Magic methods validity: resolved a false-positive (Override attribute)
- Security advisories for Composer packages: do not suggest adding roave/security-advisories by default (Composer v2.9: Automatic Security Blocking)
- Slow array function used in loop: resolved a false-positive (conditional invocation of the greedy functions)
- Variable functions usage: resolved a false-positive (array destructuring into variadic syntax transformation)
- 'in_array(...)' misused: QF code generation fix (binary expression as an argument)
