Bug-fix release focusing on gaps in PHP 8.0 to 8.4 support:

- Declare incompatibility with the ultimate version (setups with both plugins active experience varying side effects)
- Null reference: disabled (unstable, and we are limited by the platform APIs)

- Callable parameter usage violates definition: resolved a false-positive (variadic types handling)
- Exploiting unserialize (PHP Object Injection Vulnerability): resolved a false-positive (user-space function)
- Null coalescing operator can be used: resolved a false-positive (multiple cases)
- PHPUnit: bugs and best practices: correct assertion names (file/directory)
- Security advisories for Composer packages: remove symfony/var-dumper from dev-packages
- Senseless proxy function: resolved a false-positive (multiple cases)
- Slow array function used in loop: respect user-configured severity level
- Unnecessary type casting: resolved a false-positive (deeply nested null-safe operator)
- Unsupported string offset operations: resolved a false-positive (multiple cases)
