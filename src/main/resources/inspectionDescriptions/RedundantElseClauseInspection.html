Certain if-else conditions can have their else clause removed without changing the semantic. Consider the following method: 
    protected function getColumnOrValue(string $value, TableMap $table)
    {
        if (':' === $value[0]) {
            return $table->getColumn(substr($value, 1));
        } else {
            return $value;
        }
    }
In the above, the 'else' statement can be safely removed because its 'if' clause returns from the method. Thus, even without the 'else/elseif', there’s no way you’ll be able to proceed past the if clause body.

Please reference to corresponding stackoverflow thread for more details.
